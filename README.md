🧾 Multi-Step Form (Wizard Form)

A modern Multi-Step Form UI built using HTML, CSS, and JavaScript, where users fill out data step by step with validation, progress tracking, and a final review before submission.

🚀 Features
🧩 Step-by-step form completion
📊 Dynamic progress bar
✅ Form validation before moving to next step
🔙 Next & Previous navigation buttons
📋 Final summary review before submission
💾 Autosave feature using localStorage (Bonus)
✨ Smooth CSS animations between steps (Bonus)
🛠️ Technologies Used
HTML5
CSS3 (Animations & Transitions)
JavaScript (Vanilla JS)
localStorage API

⚙️ How It Works
1. Step-by-Step Form

Users complete one section at a time instead of filling all fields at once.

2. Validation

Each step is validated before moving forward to ensure required data is entered.

3. Progress Bar

A dynamic progress bar updates as the user moves through steps.

4. Final Review

Before submission, users see a summary of all entered data.

5. Autosave (Bonus)

Form data is saved in localStorage so progress is not lost on refresh.

6. Smooth Transitions

CSS animations provide smooth movement between steps.

▶️ How to Run
Download or clone the project
Open index.html in a browser
Fill out the form step-by-step
Review and submit
